# C# Fundamentals Curriculum (Pre-ASP.NET Core)

## Overview

User is learning ASP.NET Core but struggles reading C# during code review (OOP, generics, LINQ). Strong JS/React/FP background, so basic syntax (vars/loops/functions) is skipped - focus on where C# diverges from JS. Single console project (`CSharpExercises`), topic folders, `Program.cs` menu dispatcher. Early phases are fully worked examples (read+run); phase 3+ introduces TODO blanks for active implementation. See brainstorm report: `plans/reports/brainstorm-260630-2224-csharp-fundamentals-before-aspnet-core-report.md`.

Out of scope: ASP.NET Core, EF Core, web/HTTP concepts - resume those in a separate plan after this curriculum completes.
